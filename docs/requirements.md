# 功能需求规格

## 概述

History Clipboard 是一款 macOS 菜单栏剪贴板历史管理工具，自动记录用户复制的内容，支持随时回溯、搜索和再次粘贴。

## 功能需求

### FR-01：剪贴板监听
- 自动监控系统剪贴板变化
- 支持文字内容（纯文本/富文本转为纯文本存储）
- 支持图片内容（PNG/TIFF 格式）
- 去重：连续复制相同内容不重复记录

### FR-02：历史记录展示
- 菜单栏图标点击弹出历史面板
- 内容以卡片形式按时间倒序排列（最新在上）
- 置顶卡片始终显示在列表最上方
- 面板尺寸固定 360pt × 500pt

### FR-03：内容卡片
- 文字卡片：显示前 2 行文字预览，超出省略
- 图片卡片：显示缩略图预览
- 每条卡片显示复制时间（相对时间，如"3 分钟前"）
- 每条卡片右侧有置顶按钮和删除按钮

### FR-04：搜索功能
- 面板顶部提供搜索输入框
- 输入关键词实时过滤历史记录
- 搜索范围为文字内容的全文匹配

### FR-05：粘贴功能
- 点击文字卡片 → 文字写入剪贴板 → 面板收起 → 用户手动 Cmd+V 粘贴
- 点击图片卡片 → 图片写入剪贴板 → 面板收起 → 用户手动 Cmd+V 粘贴

### FR-06：置顶功能
- 可将任意卡片置顶/取消置顶
- 置顶卡片在列表顶部固定显示，时间倒序排列
- 置顶卡片不受自动清理影响

### FR-07：删除功能
- 单条删除：卡片上的删除按钮
- 删除文字记录 → 从数据库移除
- 删除图片记录 → 从数据库移除 + 删除磁盘上的图片文件

### FR-08：保留时长设置
- 提供 3 个选项：1 天 / 3 天 / 5 天
- 默认保留 3 天
- 超过保留时长的非置顶记录自动清除

### FR-09：自动清理
- App 启动时执行一次清理检查
- 运行期间每隔 1 小时执行一次后台清理
- 清理对象：超过保留时长的非置顶记录
- 图片记录清理时同时删除磁盘上的图片文件

### FR-10：开机自启
- App 安装后默认注册为登录项
- 用户登录 macOS 后自动在后台运行

### FR-11：菜单栏
- 系统菜单栏显示 App 图标
- 点击图标弹出/收起历史面板
- 右键图标可访问设置和退出选项

## 非功能需求

- NFR-01：数据库使用 SQLite，存储于 App Application Support 目录
- NFR-02：图片文件存储于 App Application Support/Images 目录
- NFR-03：剪贴板轮询间隔 0.5 秒，不影响系统性能
- NFR-04：UI 采用 Liquid Glass 设计语言，适配浅色/深色模式
